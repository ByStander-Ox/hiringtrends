About

These charts represent trends in languagues, frameworks and technologies used to develop modern software applications as indicated by Hacker News activity. Using the HN Search API and a dictionary of software terms, all of the the comments (job postings) in the "whoishiring" submissions from April 2011 to August 2013 were processed to count the number of times each software term was mentioned.

Since originally publishing, there have been a few updates to the methodology. A floor has been added of 1 mention when used to calculate YOY changes so that technologies rising from 0 or falling to 0 are captured. Additionally, only top level comments from the thread are counted in order to focus in on actual job posting comments.

Hacker News Hiring Trends (September 2013)

September's post came on a Sunday, so overall comment count was down 30% from August.

Highlights

JavaScript repeats as the top term again, while Android cracks the Top 5 and Machine Learning enters the top 10. AngularJS again tops the rising category for the fourth straight month, and while Backbone is still in the top 10, AngularJS is on track to surpass it soon (Backbone vs Angular). MySQL dropped out of the top 10, but still outpaced PostgreSQL (MySQL vs PostgreSQL). D3, a JavaScript visualization libary, also makes a 4th straight appearance on the rising list. Automation tools Puppet and Chef also see consistent growth and remain on the rising list.
